Объяснить, почему код ниже даёт именно такие результаты?
var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3
Ответы:
1) c = ++a; alert(c); // 2 - выполняется операция префиксного инкрементирования, сначала a увеличится на 1, затем с присваивается значение а.
2) d = b++; alert(d); // 1 - выполняется постфиксное инкрементирование, сначала d присвоится значение b, затем b увеличится на 1.
3) c = (2+ ++a); alert(c); // 5 - после 1-го примера a стало равно 2, затем префиксный инкримент делает его равным 3, далее складывается 2 и 3 получается 5.
4) d = (2+ b++); alert(d); // 4- после 2-го примера b стало равно 2, затем идет постфикстный инкремент в выражении он не учитывается, но далее b будет равно 3, а в итоге получается 2 + 2 = 4.
5) alert(a); // 3 - так как переменная a была инкрементирована 3 раза
6) alert(b); // 3 - так как переменная b была инкрементирована 3 раза
